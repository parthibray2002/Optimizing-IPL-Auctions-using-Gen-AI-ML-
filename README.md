# 🏏 IPL Auction 2026: Agentic AI & ML Optimization

  

## 📌 Project Overview

The IPL auction is a high-stakes marketplace where emotional bias often leads to the **"Winner’s Curse."** This project introduces an autonomous, data-driven "Digital War Room" to revolutionize player valuation for the 2026 cycle.

By leveraging a **Multi-Agentic AI framework (CoRea)**, **Ensemble Learning**, and **Deep Learning**, this system optimizes a 25-man championship-winning squad under the constraints of a **120 Crore salary cap** and strict player composition quotas.

-----

## 🚀 Key Features

  * **Predictive Valuation Engine:** Utilizes hyper-tuned **Random Forest** and **Gradient Boosting** models to classify players into price tiers (Low, Mid, High Value) with up to **97% accuracy**.
  * **Agentic AI (CoRea Framework):** A **LangChain-powered** multi-agent system featuring:
      * **Plan Agent:** Deconstructs strategic requirements into technical queries.
      * **Tool Agent:** Executes data mining and model predictions in real-time.
      * **Reflect Agent:** Acts as an autonomous auditor to ensure budget and overseas-quota compliance.
  * **Tactical Optimization:** Implements **Greedy Swap Logic** to solve the "Knapsack Problem," maximizing squad performance while adhering to fiscal guardrails.
  * **Explainable AI (XAI):** Provides "Chain of Thought" reasoning for every player recommendation, ensuring every bid is humanely justifiable.
  * **Class Balancing (SMOTE):** Addresses the scarcity of "Marquee" players by synthesizing elite performance signatures for more robust training.

-----

## 🛠️ Technology Stack

  * **Language:** Python 3.9+
  * **Agentic Framework:** LangChain (ReAct prompting)
  * **Machine Learning:** Scikit-learn (Random Forest, AdaBoost, Bagging)
  * **Deep Learning:** PyTorch (1D-CNN for tabular feature extraction)
  * **Data Processing:** Pandas, NumPy, SMOTE (imbalanced-learn)
  * **Visual Analytics:** Seaborn, Matplotlib (Interactive Dashboards)
